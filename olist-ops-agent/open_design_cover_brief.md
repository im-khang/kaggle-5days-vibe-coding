# Olist Ops Agent — Cover Image Brief

Design a 16:9 cover image for a Kaggle/GitHub capstone project: **Olist Ops Agent**.

The project is a Google ADK multi-agent analytics app for Brazilian ecommerce operations. It answers natural-language questions over BigQuery using specialist agents for fulfillment, seller operations, customer experience, payments, and data analysis. It uses SELECT-only SQL safety, tool-grounded answers, eval cases, and public setup through Vertex AI or Google AI Studio.

Visual direction: premium dark technical editorial, not generic SaaS. Show ecommerce operations as abstract logistics/data flow: delivery routes, seller/customer nodes, package movement, warehouse/data blocks, and a subtle Brazil map contour. Make it feel like serious supply-chain analytics software.

Required text hierarchy:
- Main title: **OLIST OPS AGENT**
- Secondary: **Google ADK + BigQuery analytics** — multi-agent NL queries with agents-cli eval, MCP toolsets, and Vertex AI deployment
- Small stack marks: Google ADK, agents-cli, BigQuery, MCP, Vertex AI
- Bottom coordinate strip: ADK · MULTI-AGENT · agents-cli EVAL | SELECT-ONLY SQL · TOOL-GROUNDED · 12/12 PASS | 96,999 ROWS · MCP TOOLSETS | VERTEX AI / AI STUDIO

Avoid: robots, people, fake dashboards, unreadable tiny text, cluttered badges, random logos, cartoon style, and busy text over glowing route lines.
